## Key Features

<AccordionGroup>
  <Accordion icon="zap" title="High Performance">
    - **Fast Processing**: 1:4 to 1:8 processing ratio (audio time to processing time)
    - **Real-time Streaming**: Results stream as they're generated
    - **Scalable Architecture**: Handle thousands of concurrent jobs
    - **Global CDN**: Fast access from anywhere in the world

  <Accordion icon="shield-check" title="OpenAI Compatible">

## Quick Start

Get your first transcription in under 5 minutes:

<Steps>
  <Step title="Get API Key">
    Sign up at [app.sonartext.com](https://app.sonartext.com) and generate your API key.
  </Step>
  <Step title="Make Your First Request">
    ```bash
    curl -X POST 'https://api.sonartext.com/v1/audio/transcriptions' \
      -H 'Authorization: Bearer stx_live_your_api_key' \
      -F 'file=@audio.wav' \
      -F 'model=whisper-1'
    ```
  </Step>
  <Step title="Get Results">
    Receive your transcription with timestamps, confidence scores, and detected language.
  </Step>
</Steps>

## Integration Examples

<CodeGroup>

```javascript JavaScript/Node.js
const response = await fetch('https://api.sonartext.com/v1/audio/transcriptions', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${apiKey}`
  },
  body: formData
});

const transcription = await response.json();
console.log('Transcription:', transcription.text);
```

```python Python
import requests

response = requests.post(
  'https://api.sonartext.com/v1/audio/transcriptions',
  headers={'Authorization': f'Bearer {api_key}'},
  files={'file': open('audio.wav', 'rb')},
  data={'model': 'whisper-1'}
)

transcription = response.json()
print('Transcription:', transcription['text'])
```

```php PHP
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, 'https://api.sonartext.com/v1/audio/transcriptions');
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Authorization: Bearer ' . $apiKey
]);
curl_setopt($ch, CURLOPT_POSTFIELDS, [
    'file' => new CURLFile('audio.wav'),
    'model' => 'whisper-1'
]);

$response = curl_exec($ch);
$transcription = json_decode($response, true);
```

```ruby Ruby
require 'net/http'
require 'net/http/post/multipart'

uri = URI('https://api.sonartext.com/v1/audio/transcriptions')
http = Net::HTTP.new(uri.host, uri.port)
http.use_ssl = true

request = Net::HTTP::Post::Multipart.new(
  uri,
  "file" => UploadIO.new(File.new("audio.wav"), "audio/wav", "audio.wav"),
  "model" => "whisper-1"
)
request['Authorization'] = "Bearer #{api_key}"

response = http.request(request)
transcription = JSON.parse(response.body)
```
