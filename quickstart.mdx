Get up and running with the Sonartext API in just a few minutes. This guide will walk you through making your first transcription request.

## Prerequisites

<Steps>
<Step title="Create an Account">
  Sign up for a free account at [app.sonartext.com](https://app.sonartext.com).
</Step>

<Step title="Get Your API Key">
  Navigate to the API Keys section and generate your first API key (starts with `stx_live_`).
  
  <Warning>
    Keep your API key secure and never expose it in client-side code.
  </Warning>
</Step>

<Step title="Add Credits">
  Add credits to your account to start transcribing.
</Step>
</Steps>

## Your First Transcription

<Tabs>
<Tab title="cURL">
```bash
curl -X POST 'https://api.sonartext.com/v1/audio/transcriptions' \
  -H 'Authorization: Bearer stx_live_your_api_key_here' \
  -F 'file=@audio.wav' \
  -F 'model=whisper-1' \
  -F 'response_format=verbose_json'
```
</Tab>

<Tab title="JavaScript">
```javascript
async function transcribeAudio(file) {
  const formData = new FormData();
  formData.append('file', file);
  formData.append('model', 'whisper-1');
  formData.append('response_format', 'verbose_json');

  const response = await fetch('https://api.sonartext.com/v1/audio/transcriptions', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`
    },
    body: formData
  });

  return response.json();
}
```
</Tab>

<Tab title="Python">
```python
import requests

def transcribe_audio(file_path, api_key):
    url = 'https://api.sonartext.com/v1/audio/transcriptions'
    
    headers = {'Authorization': f'Bearer {api_key}'}
    files = {'file': open(file_path, 'rb')}
    data = {'model': 'whisper-1', 'response_format': 'verbose_json'}
    
    response = requests.post(url, headers=headers, files=files, data=data)
    return response.json()
```
</Tab>
</Tabs>

## Advanced Features

### Speaker Diarization

Identify different speakers in your audio:

```javascript
const formData = new FormData();
formData.append('file', file);
formData.append('enable_diarization', 'true');
formData.append('min_speakers', '2');
formData.append('max_speakers', '4');
```

### Different Formats

Get subtitles for videos:

```bash
curl -X POST 'https://api.sonartext.com/v1/audio/transcriptions' \
  -H 'Authorization: Bearer stx_live_your_api_key_here' \
  -F 'file=@video.mp4' \
  -F 'response_format=srt' \
  -o subtitles.srt
```

## Text-to-Speech

Generate natural speech from text:

<Tabs>
<Tab title="cURL">
```bash
curl -X POST 'https://api.sonartext.com/v1/audio/speech' \
  -H 'Authorization: Bearer stx_live_your_api_key_here' \
  -H 'Content-Type: application/json' \
  -d '{
    "input": "Hello! This is generated speech.",
    "voice": "af_sky",
    "response_format": "mp3"
  }' \
  --output speech.mp3
```
</Tab>

<Tab title="JavaScript">
```javascript
async function generateSpeech(text) {
  const response = await fetch('https://api.sonartext.com/v1/audio/speech', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      input: text,
      voice: 'af_sky',
      response_format: 'mp3'
    })
  });

  return response.blob();
}
```
</Tab>
</Tabs>

## Large Files (Multipart Upload)

For files over 100MB, use multipart uploads:

```javascript
// 1. Initiate upload
const initResponse = await fetch('/v1/uploads/initiate', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${uploadToken}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    filename: file.name,
    contentType: file.type,
    totalSize: file.size
  })
});

const { uploadId, partUrls } = await initResponse.json();

// 2. Upload parts (simplified)
const parts = await uploadParts(file, partUrls);

// 3. Complete upload
await fetch('/v1/uploads/complete', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${uploadToken}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    uploadId,
    parts,
    response_format: 'verbose_json'
  })
});
```

## Job Management

Monitor long-running transcriptions:

```javascript
async function checkJobStatus(jobId) {
  const response = await fetch(`/jobs/${jobId}`, {
    headers: { 'Authorization': `Bearer ${apiKey}` }
  });
  
  const job = await response.json();
  
  if (job.status === 'completed') {
    // Get results
    const resultResponse = await fetch(`/jobs/${jobId}/result`, {
      headers: { 'Authorization': `Bearer ${apiKey}` }
    });
    return resultResponse.json();
  }
  
  return job;
}
```

## Error Handling

Always handle errors gracefully:

```javascript
try {
  const result = await transcribeAudio(file);
  console.log('Success:', result.text);
} catch (error) {
  if (error.status === 401) {
    console.error('Invalid API key');
  } else if (error.status === 402) {
    console.error('Insufficient credits');
  } else if (error.status === 429) {
    console.error('Rate limit exceeded');
  } else {
    console.error('Transcription failed:', error.message);
  }
}
```
